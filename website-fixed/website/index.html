What is ModBus Pro?

ModBus Pro is a professional Modbus diagnostic and monitoring tool designed for automation engineers, SCADA integrators, and industrial technicians. It combines Modbus master polling and slave device simulation in a single cross-platform desktop application.

The application supports Modbus RTU over RS232 and RS485 serial connections, and Modbus TCP/IP over Ethernet. It implements standard function codes including FC01 (Read Coils), FC02 (Read Discrete Inputs), FC03 (Read Holding Registers), FC04 (Read Input Registers), FC05 (Write Single Coil), FC06 (Write Single Register), FC15 (Write Multiple Coils), FC16 (Write Multiple Registers), FC22 (Mask Write Register), and FC23 (Read/Write Multiple Registers).

Whether you are commissioning a PLC, debugging an energy meter, testing a VFD drive, or verifying register maps on a BESS battery management system, ModBus Pro gives you master and slave capabilities in one window — no more switching between separate applications. Currently available for Windows 10/11, with macOS and Linux support coming soon.
